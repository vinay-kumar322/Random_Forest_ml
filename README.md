# Random Forest ML Project 

## About this project

In this project, I worked on a basic machine learning model using the Random Forest algorithm. The goal was to understand how multiple decision trees work together to improve prediction accuracy.

I used a simple dataset and built the model step by step, including data preparation, training, and testing.

---

## What I used

* Python
* Pandas
* NumPy
* Scikit-learn

---

## What I did

* Loaded the dataset
* Converted it into a DataFrame
* Split the data into training and testing sets
* Trained a Random Forest model
* Predicted results using test data
* Checked model accuracy

---

## Model details

* Algorithm: Random Forest Classifier
* Number of trees: 10
* Train-test split: 80% training, 20% testing

---

## What I learned

* How Random Forest works internally
* Difference between single tree and multiple trees
* Importance of train-test split
* How to improve model performance by tuning parameters
